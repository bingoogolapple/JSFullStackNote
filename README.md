JavaScript 全栈学习笔记
============

####初始化后端

```
npm install express formidable mongoose mongodb --save
```

####初始化前端

```
npm install webpack webpack-dev-server node-args --save-dev

npm install css-loader style-loader file-loader url-loader node-sass sass-loader --save-dev

npm install babel-loader babel-core babel-preset-es2015 --save-dev

npm install babel-preset-react react-hot-loader --save-dev


npm install react react-dom --save
npm install silly-datetime jquery --save
npm install semantic-ui --save
```
